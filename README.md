# customer_behaviour_dashboard

This end-to-end data analytics project examines customer shopping behavior to improve retail decision-making. By leveraging Python for data cleaning, SQL for exploratory analysis, and Power BI for dashboarding, it uncovers actionable insights regarding demographics, subscription impact, and revenue drivers to optimize marketing strategies."

**Project Goal**: Analyze retail data to identify high-value customers and optimize sales.

**Data Processing**: Cleaned and standardized raw data using Python (Pandas).

**Analysis**: Performed deep-dive exploratory analysis with SQL (PostgreSQL).

**Visualization**: Built an interactive Power BI dashboard to track KPIs and trends.

# Technologies Used

**Python**: Pandas (Data Cleaning, Feature Engineering)

**SQL (PostgreSQL)**: Advanced Data Analysis, Complex Queries

**Power BI**: Data Visualization, Dashboarding, DAX

**IDLE Python:** For Python scripting
